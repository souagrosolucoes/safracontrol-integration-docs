# Bem-vindo à documentação da API

Esta documentação cobre todos os endpoints disponíveis, fluxos de uso
e exemplos de integração.

## Por onde começar

1. Leia o [Guia rápido](./guia-rapido.md) para ter a API funcionando em minutos
2. Entenda o [Fluxo de autenticação](./fluxos/autenticacao.md) antes de fazer qualquer requisição
3. Explore os endpoints na referência completa à esquerda
